Floor function

Last Updated: 28 February 2023

The floor function is a function \(floor: \mathbb R \rightarrow \mathbb Z\) that rounds down a real number to the nearest integer. To be specific, \(floor(x)\), abbreviated as \(\lfloor {x}\rfloor \), returns the largest integer \(y\) such that \(y \leq x\). The floor function's counterpart is the ceiling function.

For example, \(\lfloor {7.83}\rfloor = 7\), \(\lfloor {5}\rfloor = 5\), \(\lfloor {-2.2}\rfloor = -3\) and \(\lfloor {-9}\rfloor = -9\).

⚠ When the floor function is given an integer as its input, it returns the exact same integer as its output.
